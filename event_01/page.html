리워드 받기
  • * 신한 슈퍼SOL 가입 시 권유직원 등록에서 '신한라이프' 선택 필수
  • * 슈퍼SOL 가입 후 '리워드 받기' 버튼 클릭 필수

※ 이벤트 유의사항

  • 참여대상 : 신한 슈퍼SOL 앱 신규 가입자
  • 참여방법 : 신한 슈퍼SOL 앱 가입 후 '리워드 받기' 버튼 클릭(가입시 권유직원 '신한라이프' 선택 필수)
  • 신한 슈퍼SOL 최초 가입 시 '신한라이프' 을 반드시 선택해야 하며, 가입 후 재입력 혹은 수정은 불가합니다.
  • 신한 슈퍼SOL 가입 후 해당 월의 말일까지 '리워드 받기' 버튼을 클릭해야 리워드가 지급됩니다.
  • '리워드 받기' 버튼을 누르는 즉시 리워드가 지급됩니다.
  • 리워드 지급 대상자임에도 불구하고 리워드를 받지 못한 경우 신한라이프 고객지원센터(02-3772-2220)로 문의 부탁드립니다
  • 리워드 지급은 1인 1회 제공을 원칙으로 합니다. 다 회 참여 시 1회만 인정됩니다.
